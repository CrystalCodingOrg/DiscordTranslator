üìù What Data We Collect & Store

When you use our translation service, we automatically collect and store:

  • Discord User ID: Your unique Discord identifier (numeric ID)
  • Username: Your Discord username at the time of translation
  • Original Messages: The full text of messages you submit for translation
  • Message Hashes: SHA-256 hashes of original messages for efficient caching
  • Translated Messages: The complete translated output
  • Target Languages: Which languages you request translations into
  • Detected Languages: The source language detected by our AI
  • Timestamps: When each translation was created and last accessed
  • Usage Counts: How many times each cached translation has been reused

üéØ Why We Collect This Data

  • Performance Caching: Store frequently translated messages to provide instant responses without re-processing
  • Cost Optimization: Reduce API calls to translation services by reusing cached results
  • User Tracking: Link translations to users for accountability and usage analytics
